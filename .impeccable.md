# Мясорубка — Telegram Mini App

## Design Context

### Users
Маркетинговые специалисты в РФ/СНГ: директологи, таргетологи, авитологи, дизайнеры, лендинг-спецы. Чаще фрилансеры или спецы на загрузке проектами. Контекст использования — телефон, внутри Telegram, эмоциональное состояние «меня опять сделали крайним за чужой говнобриф». Это не B2B-энтерпрайз, это люди, которых задело за живое. Продукт — воронка-игра, ведущая к покупке трипваера «Лендос за вечер» (3999₽).

### Brand Personality
Три слова: **дерзкий, киберспортивно-турнирный, безжалостно-честный.**
- Голос: прямой, без воды, говорит на языке спеца («говнобриф», «слил в дырку», «крайний»). Тыкает в боль, но не унижает — встаёт на сторону спеца против системы.
- Эмоции по ходу: узнавание → удар (defeat) → катарсис → надежда → желание купить.
- Метафора: грязный маркетинговый ranked-режим. Проект = матч. Клеймо = defeat screen. Лупа = replay review. AI-штаб = skill unlock.

### Aesthetic Direction
**Elo Hell — premium dark gaming-ranked.** НЕ буквальный геймерский HUD (пиксели, джойстики, аниме), а визуальные ПАТТЕРНЫ ранговых систем (ранг, матч найден, defeat, MMR, promotion failed, replay) поданные дорого и кинематографично.
- Тема: **dark** — выбор оправдан контекстом (вечерний телефон, драматургия поражения, киберспорт-эстетика).
- Сейчас палитра: bg #080810, red #E63946, gold #FFB703, blue #4361EE. Работает, но базовая — нужно углубить через OKLCH, тинт нейтралей, премиальные тени/градиенты.
- Референсы по вайбу: defeat-экраны League of Legends/Valorant, rank-up анимации, дорогие dark-лендинги Linear/Vercel/Igloo Inc, кинематографичные reveal'ы.
- Анти-референсы: дешёвый неон-на-чёрном «AI slop», фиолетово-синие градиенты, glassmorphism везде, generic карточки с иконкой сверху, gradient text.

### Design Principles
1. **Удар важнее украшения.** Главные вау-моменты (печать «ТЫ КРАЙНИЙ», лупа, defeat-метрики) должны бить кинематографично — оркестрованные reveal'ы, а не россыпь микроанимаций.
2. **Дорого = выверенно, не кричаще.** Премиум создаётся точностью: тайминги, easing (ease-out-expo/quint), глубина теней, тинтованные нейтрали, благородные градиенты-подложки — а не количеством эффектов.
3. **Тактильность.** Телега даёт хаптику — каждый ключевой момент должен ощущаться телом, не только глазами. Motion + haptics синхронны.
4. **Турнирная драматургия.** Поток экранов — это матч с актами: лобби → пик → запуск → defeat → разбор → апгрейд. Переходы должны поддерживать нарратив, а не быть просто слайдами.
5. **Узнаваемость без буквальности.** Берём грамматику киберспорта (бейджи, шкалы рангов, defeat-затемнение), но исполняем как премиум-продукт, чтобы не превратить боль в геймерский прикол.
